Adding my google photos

16 Apr 2021 Pseudo Surfer

Rather than host images locally I’ve found a way of adding my Google Photos into my site.

I’m using Haltakov’s simple-photo-gallery. It’s a script written in Python which pulls all the links to all the photos of a given gallery and builds static files. Getting the static files into Jekyll was interesting since I wanted to add front matter and keep an organised list of galleries in my _data folder so that I can add some more organisation down the road. One of the nice things about his script is that I can edit the images_data.json file and add descriptions to each photo if I wish to. The gallery can then be rebuilt and I just need to copy the linking portion of the html files and update my local files.
